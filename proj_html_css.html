1er Projet HTML/CSS — Les reptiles

Site réalisé pour apprendre les bases du HTML et CSS.

Résumé

Site pédagogique sur des reptiles rares, développé pour apprendre les bases du HTML et CSS et la structuration de contenu.

Contexte

Projet scolaire en 1ère année de BTS SIO visant à prendre en main les technologies front-end (HTML, CSS).

Besoins (grille E5)

  • Présenter du contenu pédagogique de manière claire et accessible.
  • Respecter une structure HTML sémantique et du style CSS responsive.

Actions réalisées

  1. Conception de la structure HTML et des sections de contenu.
  2. Réalisation des styles CSS (mise en page, responsive simple).
  3. Optimisation des images et tests multi-écrans.

Résultats

Site en ligne, lisible sur mobiles et desktop, amélioration des compétences de markup et layout.

Compétences mobilisées (grille E5)

  • Structurer une page web (balises sémantiques).
  • Appliquer des règles de présentation CSS.
  • Tester et corriger l'affichage responsive.

Outils

Éditeur de texte, navigateur pour tests, GitHub Pages pour hébergement.

Rôle & durée

Développeur seul — 2 semaines (exemple).
